Usually, when talking about design systems, we start with our Figma library, component library, and design tokens. We reduce it to a collection of artifacts. We think it is a tidy library of buttons, inputs, and colors that sit nicely in a repository. It feels good to organize these things. It feels like control. It is clean, organized, and theoretical.

But that feeling is an illusion.
