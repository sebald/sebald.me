```css
--shadow-cutout: 0 1px 0 0 oklch(from var(--color-oatmeal-400) l c h / 0.2),
  inset 0 1px 0 0 oklch(from var(--color-oatmeal-400) l c h / 0.15),
  inset 0 -1px 0 0 oklch(from var(--color-oatmeal-950) l c h / 0.6);
```

## Visual Breakdown

This "cutout" effect works by simulating light interaction with an indented surface:

| Layer              | Type  | Offset   | Direction        | Role          | Visual Result                                    |
| :----------------- | :---- | :------- | :--------------- | :------------ | :----------------------------------------------- |
| **Surface Lip**    | Outer | `0 1px`  | Bottom (Outside) | **Highlight** | Light hitting the bottom edge of the "hole."     |
| **Top Bevel**      | Inset | `0 1px`  | Top (Inside)     | **Specular**  | Light catching the top rim of the object inside. |
| **Bottom Crevice** | Inset | `0 -1px` | Bottom (Inside)  | **Shadow**    | The dark gap where the object meets the floor.   |
